# URL Shortener Service

A production-like URL shortening REST service built with Java 21 and Spring Boot 3.

## Status

The project is under active development. The base Maven structure and application configuration are in place.

## Planned capabilities

- Create and manage short links
- Redirect through generated or custom aliases
- PostgreSQL persistence and Flyway migrations
- Redis caching with expiration-aware TTL
- Click analytics and daily statistics
- OpenAPI documentation and Swagger UI
- Unit, controller, and Testcontainers integration tests
- Docker Compose environment
- Minimal static web interface

## Technology stack

Java 21, Spring Boot, Spring Web, Spring Data JPA, PostgreSQL, Redis, Flyway, Maven, JUnit 5, Mockito, Testcontainers, Docker, and springdoc-openapi.
